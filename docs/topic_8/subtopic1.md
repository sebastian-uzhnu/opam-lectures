# Масиви

Масив - це структура даних яка складається з впорядкованого набору фіксованої кількості однотипних елементів, що
зберігаються в послідовно розташованих комірках оперативної пам'яті, мають порядковий номер і спільне ім'я, що надає
користувач.
Кожен елемент масиву має свій індекс, починаючи з 0. Масиви дозволяють ефективно зберігати та обробляти дані.

> Масиви є важливими структурами даних, вони корисні для організації і обробки наборів даних.

## Оголошення та ініціалізація масивів

### Оголошення масиву:

```csharp
int[] numbers;  // Оголошення масиву цілих чисел
```

### Ініціалізація масиву:

```csharp
numbers = new int[5];  // Створення масиву з 5 елементів
```

### Оголошення та ініціалізація масиву одночасно:

```csharp
int[] numbers = new int[] { 1, 2, 3, 4, 5 };  // Оголошення і ініціалізація масиву
```

### Оголошення і ініціалізація масиву зі значеннями:

```csharp
int[] numbers = { 1, 2, 3, 4, 5 };  // Ініціалізація масиву з 5 елементів
```

### Оголошення масиву без ініціалізації значень:

```csharp
int[] numbers = new int[5];  // Створення масиву з 5 елементів, всі значення будуть 0
```

### Масиви об'єктів:

```csharp
string[] names = new string[] { "Alice", "Bob", "Charlie" };  // Масив рядків
```

## Доступ до елементів масиву

Доступ до елементів масиву здійснюється за допомогою індексів, які починаються з нуля.

```csharp
int[] numbers = { 1, 2, 3, 4, 5 };
int firstNumber = numbers[0];  // Доступ до першого елемента (1)
numbers[2] = 10;  // Зміна значення третього елемента на 10
```
