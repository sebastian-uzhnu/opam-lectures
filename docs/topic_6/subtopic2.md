# Таблиці істинності

### "І" (AND)

| A     | B     | A && B |
|-------|-------|--------|
| true  | true  | true   |
| true  | false | false  |
| false | true  | false  |
| false | false | false  |

### "АБО" (OR)

| A     | B     | A \|\| B |
|-------|-------|----------|
| true  | true  | true     |
| true  | false | true     |
| false | true  | true     |
| false | false | false    |

### "НЕ" (NOT)

| A     | !A    |
|-------|-------|
| true  | false |
| false | true  |

### "Виключне АБО" (XOR)

| A     | B     | A ^ B |
|-------|-------|-------|
| true  | true  | false |
| true  | false | true  |
| false | true  | true  |
| false | false | false |
