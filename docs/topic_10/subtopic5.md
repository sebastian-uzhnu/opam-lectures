# Файлова система

Для роботи з файловою системою в C# використовується простір імен `System.IO`. Він містить класи для роботи з файлами, папками та потоками даних.

## Основні класи System.IO

* `File`: статичний клас для створення, копіювання, видалення, переміщення та відкриття файлів.
* `FileInfo`: надає методи екземпляра для створення, копіювання, видалення, переміщення та відкриття файлів.
* `Directory`: статичний клас для створення, переміщення та перерахування каталогів і підкаталогів.
* `DirectoryInfo`: надає методи екземпляра для роботи з каталогами.
* `Path`: виконує операції з рядками шляхів до файлів або каталогів.

## Абсолютний і відносний шлях

* **Абсолютний шлях**: повний шлях від кореневого каталогу (наприклад, `C:\Users\User\Documents\file.txt`).
* **Відносний шлях**: шлях відносно поточної робочої директорії програми (наприклад, `data\file.txt`).

```csharp
// Отримання поточного каталогу
string currentDir = Directory.GetCurrentDirectory();

// Об'єднання шляхів (кросплатформно)
string fullPath = Path.Combine(currentDir, "data", "report.txt");
```

## Перевірка існування

Перед роботою з файлом або папкою варто перевірити їх наявність:

```csharp
if (File.Exists("config.txt"))
{
    // Файл існує
}

if (Directory.Exists("logs"))
{
    // Папка існує
}
```

## Обробка помилок доступу

Робота з файлами часто супроводжується винятками (файл не знайдено, немає прав доступу, файл зайнятий іншим процесом). Тому код для роботи з файлами слід поміщати в блок `try-catch`.

```csharp
try
{
    string content = File.ReadAllText("secret.txt");
}
catch (FileNotFoundException ex)
{
    Console.WriteLine("Файл не знайдено!");
}
catch (UnauthorizedAccessException ex)
{
    Console.WriteLine("Немає прав доступу!");
}
catch (Exception ex)
{
    Console.WriteLine($"Сталася помилка: {ex.Message}");
}
```
