# Створення та ініціалізація рядків

У C# рядок — це об'єкт типу `System.String`, який представляє послідовність символів Unicode. Ключове слово `string` є псевдонімом для класу `System.String`.

## Оголошення та ініціалізація

Існує кілька способів створення рядків:

```csharp
// Оголошення без ініціалізації (не рекомендується використовувати без присвоєння)
string message; 

// Ініціалізація літералом
string greeting = "Hello, World!";

// Порожній рядок
string empty = "";
string alsoEmpty = String.Empty;

// Ініціалізація null (відсутність об'єкта)
string nullString = null;

// Створення з масиву символів
char[] letters = { 'H', 'e', 'l', 'l', 'o' };
string fromChars = new string(letters);
```

## Escape-послідовності

Для використання спеціальних символів у рядках використовуються escape-послідовності, що починаються з `\`:

* `\n` — новий рядок
* `\t` — табуляція
* `\"` — подвійні лапки
* `\\` — зворотний сліш

```csharp
string path = "C:\\Windows\\System32";
string quote = "Він сказав: \"Привіт!\"";
```

## Verbatim-рядки (Дослівні рядки)

Якщо перед рядком поставити символ `@`, то escape-послідовності ігноруються. Це зручно для шляхів до файлів:

```csharp
string path = @"C:\Windows\System32";
string multiline = @"Це рядок,
який займає
кілька ліній коду.";
```

## Інтерполяція рядків

Починаючи з C# 6.0, можна використовувати інтерполяцію (символ `$`) для вставки значень змінних прямо в рядок:

```csharp
string name = "Олексій";
int age = 20;
string info = $"Ім'я: {name}, Вік: {age}"; // "Ім'я: Олексій, Вік: 20"
```
