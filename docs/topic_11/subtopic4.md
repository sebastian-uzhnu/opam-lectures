# Основні принципи ООП на реальних прикладах

## Поліморфізм

**Одна команда — різні дії**

- Кнопка «Play»: телефон → музика, YouTube → відео, пульт → телевізор
- Команда «їхати»: авто → на колесах, поїзд → по рейках, літак → летить

## Інкапсуляція

**Всередині сховано, зовні — кнопки**

- Банкомат: бачиш екран і кнопки, не бачиш механізмів
- Пральна машина: вибір режиму, але не процес прання

## Наслідування

**Дитина бере властивості від батьків**

- Авто → електроавто (колеса, кермо, гальма)
- Телефон → смартфон (екран, дзвінки, камера)

## Абстракція (тільки важливе)

- Кермо: повертати авто, без деталей двигуна
- Кнопка «On/Off»: вмикає пристрій, як — неважливо

---

## 1 речення для запам’ятовування

- **Поліморфізм** — одна дія, різний результат
- **Інкапсуляція** — приховані деталі
- **Наслідування** — спільні властивості
- **Абстракція** — тільки важливе
