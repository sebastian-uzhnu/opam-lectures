# Структури в C#


Структури (struct) та об’єднання — це спеціальні типи даних, які дозволяють об’єднувати кілька змінних різних типів в одну логічну групу. У C# існують лише **структури** (об’єднань в C# немає, хоча в деяких інших мовах програмування, таких як C або C++, вони існують). У структурах зберігаються дані як значимі типи, тобто значення зберігаються безпосередньо в змінних, а не як посилання на об’єкти в купі.


Структура — це визначений користувачем тип, який використовується для групування кількох пов'язаних змінних, що можуть бути різних типів. Структури корисні тоді, коли необхідно зберігати невеликі об’єкти, які не вимагають складної поведінки, як, наприклад, для зберігання точок або відстаней.

### Визначення структури:

```csharp
public struct Point
{
    public int X;
    public int Y;

    public Point(int x, int y)
    {
        X = x;
        Y = y;
    }
}

```

### Створення та використання структур:

```csharp
Point point = new Point(5, 10);
Console.WriteLine($"X: {point.X}, Y: {point.Y}");

```

У цьому прикладі структура `Point` містить два поля `X` та `Y`, які представляють координати точки. Значення цих полів задаються через конструктор.

### Порівняння структур і класів

Основна різниця між **структурами** та **класами** в C# полягає в тому, що структури є значимими типами, тоді як класи є посилальними типами:

- **Значимі типи (структури)** зберігають свої значення безпосередньо у змінних. Якщо змінну зі структурою передати в інший метод, її копія буде створена.
- **Посилальні типи (класи)** зберігають лише посилання на об'єкт в купі. Якщо передати посилання на клас в інший метод, зміни в ньому впливатимуть на оригінальний об'єкт.

### Порівняння структур і класів:

| Параметр | Структури | Класи |
| --- | --- | --- |
| **Тип** | Значимий | Посилальний |
| **Зберігання даних** | На стеку | В купі |
| **Передача в методи** | Копіювання | Передача посилання |
| **Наслідування** | Не підтримується | Підтримується |
| **Продуктивність** | Швидше (для невеликих даних) | Повільніше для малих об'єктів |

### Способи доступу до вмісту структур

- Доступ до полів структури можна здійснити через ім'я змінної та поля.

### Приклад доступу до полів структури:

```csharp
Point point = new Point(2, 3);
Console.WriteLine(point.X);  // Виведе 2
point.Y = 10;
Console.WriteLine(point.Y);  // Виведе 10

```

Структури підтримують **методи, конструктори, властивості, інтерфейси**, але вони не можуть мати деструкторів або використовувати успадкування.
