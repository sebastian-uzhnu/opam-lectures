# Консольні програми

Консольні програми є базовими програмами, що виконуються у командному рядку або терміналі, без графічного інтерфейсу
користувача. Вони є відмінним способом для новачків ознайомитися з основами програмування на C#, оскільки дозволяють
зосередитися на базових концепціях мови без додаткових складнощів графічного інтерфейсу.

## Взаємодія з користувачем

Консольні програми часто використовують клас `Console` для взаємодії з користувачем. Методи класу `Console` дозволяють
виводити текст на екран і зчитувати ввід з клавіатури.

**Основні методи класу `Console`:**

- **`Console.WriteLine()`**: Виводить текст на новому рядку.
- **`Console.Write()`**: Виводить текст без переносу рядка.
- **`Console.ReadLine()`**: Зчитує рядок з вводу користувача.
- **`Console.Read()`**: Зчитує один символ з вводу користувача.

## Приклад взаємодії з користувачем:

```csharp
using System;

class Program
{
    static void Main(string[] args)
    {
        Console.Write("Enter your name: ");
        string name = Console.ReadLine();  // Зчитування вводу користувача
        Console.WriteLine($"Hello, {name}!");  // Виведення привітання
    }
}
```

## Аргументи командного рядка

Метод `Main` може приймати параметри командного рядка. Це масив рядків `string[] args`, в якому зберігаються аргументи,
передані при запуску програми.

## Приклад обробки аргументів командного рядка:

```csharp
using System;

class Program
{
    static void Main(string[] args)
    {
        if (args.Length > 0)
        {
            Console.WriteLine("Arguments provided:");
            foreach (string arg in args)
            {
                Console.WriteLine(arg);
            }
        }
        else
        {
            Console.WriteLine("No arguments provided.");
        }
    }
}
```

**Запуск програми з аргументами:**

```bash
dotnet run arg1 arg2 arg3
```

## Форматування виводу

Клас `Console` підтримує різні методи форматування виводу, включаючи вставку змінних в рядки. Ви можете використовувати
рядкові інтерполяції або форматування рядків для контролю вигляду виводу.

## Приклад форматування рядків:

```csharp
using System;

class Program
{
    static void Main(string[] args)
    {
        int age = 25;
        string name = "Alice";

        // Рядкова інтерполяція
        Console.WriteLine($"Name: {name}, Age: {age}");

        // Форматування рядків
        Console.WriteLine("Name: {0}, Age: {1}", name, age);
    }
}
```

## Обробка помилок

Консольні програми можуть використовувати обробку виключень для управління помилками, які можуть виникати під час
виконання програми.

## Приклад обробки виключень:

```csharp
using System;

class Program
{
    static void Main(string[] args)
    {
        try
        {
            Console.Write("Enter a number: ");
            int number = int.Parse(Console.ReadLine());  // Може викликати FormatException
            Console.WriteLine($"You entered: {number}");
        }
        catch (FormatException ex)
        {
            Console.WriteLine("Invalid input. Please enter a valid number.");
        }
    }
}
```
