# Вступ

У процесі розробки програм часто виникає потреба описати загальну поведінку для групи об'єктів, залишаючи конкретну реалізацію для похідних класів.

Для цього в C# використовуються:

- Абстрактні класи
- Інтерфейси

Ці механізми дозволяють будувати гнучку, розширювану та підтримувану архітектуру програм.
